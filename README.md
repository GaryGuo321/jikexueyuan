# jikexueyuan  

生产环境：[点击这里](http://garyguo321.github.io/jikexueyuan/dist/)  
开发环境：[点击这里](http://garyguo321.github.io/jikexueyuan/src/)  

安装node依赖：
```
npm install
```

启动项目(开发环境)：
```
npm start 或 gulp server
```

启动项目(dist环境)：
```
npm test
```

构建项目：
```
npm build 或 gulp
```
